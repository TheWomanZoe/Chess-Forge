# ChessForge - алгоритъм за симулирането на човешката игра

### Линк към платформата: [https://chess-forge.vercel.app](https://chess-forge.vercel.app)

ChessForge е компютър, създаден да симулира човешката игра в шаха. Използва Minimax алгоритъм с Alpha-Beta Pruning за намиране на най-добрия ход. Дълбочината на търсене се определя от времето, с което разполага за всеки ход.

Създаден е поради моя личен интерес към шаха и компютрите в него :-). Целта ми е да разработя алгоритъм, който да мисли и прави грешки по подобен начин на човека, вместо да играе перфектно или да вкарва нарочно неточности.

### Можете да използвате платформата за изпробване на алгоритъма или да създадете локален сървър, като клонирате това repository и следвате инструкциите:

1. Инсталиране на project dependencies ```npm install``` 
2. Създаване на локален сървър ```npm run dev```
